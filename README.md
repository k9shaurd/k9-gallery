A native Android application written in Kotlin that displays a curated collection of artworks or books in a clean, minimalist interface. The app focuses on handling high-quality images and seamless navigation between screens.

* **UI:** XML Layouts & ViewBinding
* **Image Loading:** [Coil](https://coil-kt.github.io/coil/)
* **Architecture:** MVVM (Simplified)
* **Data Passing:** Android `Parcelable` (API 33+ compatible)
